# Adaptive Study Planner

An AI-based adaptive study planner that generates personalized learning paths
using A* search, CSP scheduling, and LLM-based curriculum generation.

## Features
- LLM-based curriculum generation
- A* search for optimal topic ordering
- Constraint-based scheduling
- Adaptive replanning
- Streamlit UI

## Tech Stack
Python, Streamlit, NetworkX, SQLite, A*, CSP, Gemini LLM

## Run Locally
streamlit run app.py
